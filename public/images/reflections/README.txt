Reflection artwork goes here, named per the Brand Guide convention: BC-001.png, RT-001.png,
WB-001.png, FFH-001.png, BBW-001.png.

A file alone does nothing. Set the matching record"s `artwork` field in data/reflections.ts:
    artwork: "/images/reflections/BC-001.png"

Until then the reflection renders as a typeset plate tinted from its own Suggested Colour
Palette. That is by design, not a placeholder to be embarrassed about.
